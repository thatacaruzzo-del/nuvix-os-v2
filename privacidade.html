Sua privacidade é nossa prioridade. Esta política explica de forma clara e transparente como tratamos seus dados no Nuvix Hub, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

1. Quem somos

O Nuvix Hub é uma plataforma SaaS (Software como Serviço) de gestão empresarial desenvolvida para empresas de serviços, transporte e comércio.

Somos responsáveis pelo tratamento dos dados pessoais coletados através de nossa plataforma e nos comprometemos com a transparência e segurança das informações.

  • Produto: Nuvix Hub — Sistema Operacional Empresarial
  • Contato: contato@nuvixhub.com.br
  • Encarregado de dados (DPO): contato@nuvixhub.com.br

2. Dados que coletamos

2.1 Dados da empresa cliente

  • Identificação: Razão social, nome fantasia, CNPJ
  • Contato: E-mail, telefone, endereço
  • Plano e pagamento: Plano contratado, data de vencimento

2.2 Dados dos usuários

  • Identificação: Nome completo, e-mail, cargo
  • Acesso: Último acesso, IP de acesso, navegador utilizado
  • Credenciais: Senha (armazenada de forma segura)

2.3 Dados operacionais

  • Dados financeiros lançados pelo cliente (receitas, despesas)
  • Dados de clientes cadastrados pelo usuário
  • Dados de colaboradores (RH, folha de pagamento)
  • Dados de transporte e logística
  • Dados de materiais e estoque
Os dados operacionais pertencem ao cliente. O Nuvix Hub atua como operador desses dados, seguindo as instruções do controlador (a empresa cliente).

3. Como usamos os dados

  • Prestação do serviço: Operar e manter a plataforma funcionando
  • Autenticação: Identificar e autenticar usuários no sistema
  • Suporte: Responder dúvidas e resolver problemas técnicos
  • Segurança: Detectar e prevenir acessos não autorizados
  • Melhorias: Aprimorar funcionalidades da plataforma
  • Comunicação: Enviar avisos sobre o serviço, atualizações e vencimentos
  • Cobrança: Gerenciar assinaturas e pagamentos

Não utilizamos seus dados para venda a terceiros, publicidade de outros serviços ou qualquer finalidade não descrita nesta política.

4. Base legal para tratamento

Tratamos seus dados com base nas seguintes hipóteses legais previstas na LGPD:

  • Execução de contrato — para prestar o serviço contratado
  • Legítimo interesse — para segurança e melhoria do sistema
  • Cumprimento de obrigação legal — quando exigido por lei
  • Consentimento — para comunicações de marketing (quando aplicável)

5. Compartilhamento de dados

Seus dados não são vendidos a terceiros. Podemos compartilhar informações apenas com:

  • Supabase (banco de dados): Infraestrutura de armazenamento — Política de privacidade
  • Vercel (hospedagem): Servidor da aplicação — Política de privacidade
  • Autoridades: Quando exigido por lei ou ordem judicial
Todos os nossos fornecedores são contratados com cláusulas de proteção de dados e operam em conformidade com as legislações aplicáveis.

6. Armazenamento e segurança

  • Armazenamento: Dados armazenados em servidores seguros com criptografia
  • Controle de acesso: Cada empresa acessa apenas seus próprios dados (isolamento por company_id)
  • RLS: Row Level Security ativado em todas as tabelas do banco de dados
  • Senhas: Armazenadas com proteção adequada
  • HTTPS: Toda comunicação é criptografada via SSL/TLS
  • Retenção: Dados mantidos enquanto o contrato estiver ativo + 5 anos após encerramento

Em caso de incidente de segurança que possa afetar seus dados, notificaremos você e a ANPD dentro dos prazos legais.

7. Seus direitos (LGPD)

Como titular dos dados, você tem os seguintes direitos garantidos pela LGPD:

  • Acesso: Saber quais dados temos sobre você
  • Correção: Corrigir dados incompletos ou desatualizados
  • Exclusão: Solicitar a exclusão dos seus dados
  • Portabilidade: Receber seus dados em formato estruturado
  • Revogação: Revogar consentimento a qualquer momento
  • Oposição: Opor-se ao tratamento em determinadas situações
  • Informação: Ser informado sobre com quem compartilhamos seus dados

Para exercer qualquer um desses direitos, entre em contato pelo e-mail: privacidade@nuvixhub.com.br

Responderemos em até 15 dias úteis.

8. Cookies

O Nuvix Hub utiliza session storage (armazenamento de sessão) para manter você autenticado durante o uso. Esse dado é eliminado automaticamente ao fechar o navegador.

Não utilizamos cookies de rastreamento, publicidade ou analytics de terceiros.

9. Menores de idade

O Nuvix Hub é uma plataforma voltada exclusivamente para uso empresarial. Não coletamos intencionalmente dados de menores de 18 anos.

Se identificarmos que dados de menores foram coletados sem consentimento adequado, procederemos com a exclusão imediata.

10. Alterações desta política

Esta política pode ser atualizada periodicamente. Em caso de alterações significativas, notificaremos os usuários por e-mail e/ou aviso na plataforma com 30 dias de antecedência.

A continuidade do uso após as alterações implica na aceitação da nova política.
